PRS order helper

This tool will help make the comment text in PRS quicker

  • It uses a template spreadsheet with your information and gives you a text file
  • You can't change the names for the column headers, but you can move them around
  • Upload your spreadsheet using the upload button
  • Press submit to get your text file
